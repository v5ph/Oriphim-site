    Privacy Policy

Sections to Include:

Information Collected

Describe what data is collected:

Email, name, IP address, usage logs (if applicable)

Stripe billing data (handled securely via third-party processor)

Use of Data

Explain use for account access, API delivery, technical support, billing, and fraud prevention

Data Sharing

State that no client data is sold or shared with third parties beyond infrastructure providers (e.g. Stripe, Supabase)

Security

Explain that industry-standard security protocols (TLS, key rotation, access control) are used to protect data

Client Confidentiality

Include a clause noting that strategy names, signal access, and usage patterns are treated as confidential

Cookies & Tracking

Disclose basic analytics (e.g., Plausible or privacy-safe tracking if used)

Data Retention

Define how long user data is retained (e.g. 12 months after account closure)

Your Rights

State how clients can request data deletion, changes, or review under applicable laws (e.g. CCPA/GDPR if relevant)
